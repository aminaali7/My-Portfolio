Projects

Mini javascript tasks

1. Colour flipper

The first project is a site called colour flipper, where the four buttons correspond with the background colour of the site. So when the red button is pressed the background turns red and so on and so forth. The buttton labelled random generates any possible outcome from the colour wheel when pressed.

2. Palindrome checker

The second project is a site called palindrome checker, where there is an input bar to type in and the button labelled 'Check!' determines if the word inputed is a palindrome or not - and notifies the user accordingly.

3. Random quote generator

The third project is a simpler site called random quote generator. Pressing the button labelled 'quote' generates a random quote from a selection pasted into the jaavscript.

4. Stopclock

The fourth project is a stopclock site; complete withe a stop, reset and start button and a typical digital clock format.

5. Todo list

The last javascript task is a simple todo list where there is an input bar to type in and add items, the items added will then be showcased below the bar in a list format, and are able to be deleted.
